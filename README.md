# dotfiles

Dot files and setup scripts for Mac.

## Install

1. Download this repository to your Mac.

    ```sh
    git clone https://github.com/furusax0621/dotfiles.git $HOME/dotfiles
    ```
1. Run shell scripts.

   ```sh
   cd ~/dotfiles/
   ./macos.sh
   ./terminal.sh
   ./editor.sh
   ```
1. Restart your Mac.
